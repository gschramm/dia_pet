# Demo PET simulations

## Installation

Using conda
```
conda env create -f environment.yaml
```

Using mamba (recommended)
```
mamba env create -f environment.yaml
```

**If you don't have a CUDA GPU, remove the cupy dependency from environment.yaml**

## Run

```
python mlem.py -h
```
